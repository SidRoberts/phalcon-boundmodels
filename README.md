Sid\Phalcon\BoundModels
=======================

Automatically get models based on dispatcher parameters within the Phalcon framework.

## Installing ##

Install using Composer:

```json
{
    "require": {
        "sidroberts/phalcon-boundmodels": "dev-master"
    }
}
```
